# DIO-AWS-stepfunction
Desafio da DIO com o propósito de falar sobre o AWS Step Functions, suas anotações e insights adquiridos durante a prática do curso

AWS Step Functions resumidamente se trata de um construtor visual para criar fluxos de trabalho.
Mais especificamente ele é usado para desenvolver aplicações distribuídas, automatizar processos, orquestrar microsserviços e criar pipelines de dados e machine learning (ML).

Casos de uso: 
Cadastro de usuário > validar dados > criar conta > enviar e-mail.
Pedido de loja > pagamento > atualização de estoque > confirmar compra.
Upload de imagem → redimensionar → salvar → notificar o usuário.
